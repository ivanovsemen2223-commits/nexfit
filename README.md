# NEXFIT — карточка товара «Hier»

🔗 Демо: https://nexfit-nu.vercel.app

Карточка товара интернет-магазина спортивной одежды: мужской костюм «Hier», худи плюс шорты. Сделано как живая страница из крупного магазина — со всеми механиками выбора и покупки, но с упором на одну сильную скролл-сцену, которая держит внимание. Целит в аудиторию, которая привыкла к маркетплейсам и сразу ищет размер, цвет и кнопку.

Верх экрана — рабочая карточка: галерея со стрелками, превью и плавной сменой кадра, выбор цвета свотчами (название цвета подхватывается на лету), размеры, степпер количества, «в корзину» со счётчиком и пульсом на бейдже, кнопка «в избранное». Под ней табы «Описание / Характеристики / Уход». Дальше идёт фирменная пин-секция «СОЗДАН ДЛЯ ДВИЖЕНИЯ»: слова собираются снизу, окно с фото раскрывается на весь экран через clip-path, заголовок улетает, проявляется подпись — вся сцена привязана к прогрессу скролла, так что всегда доигрывает до конца, как быстро ни крути.

Дальше — описание, где слова проявляются по одному в стиле Apple, лента «то, что вы смотрели» со стрелками, преимущества 01–03, отзывы с рейтингом и, в подвале, гигантская надпись nexfit, подогнанная по ширине экрана, у которой буквы выезжают по одной. Анимации построены на GSAP ScrollTrigger поверх плавного скролла Lenis, с честным учётом prefers-reduced-motion.

Тех: HTML/CSS/JS, пин-сцена на GSAP ScrollTrigger (clip-path reveal), плавный скролл Lenis, галерея/табы/счётчик корзины руками, шрифты Inter / Comfortaa.
