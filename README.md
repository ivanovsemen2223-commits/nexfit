# NEXFIT — лендинг спортивного костюма «Hier»

Продуктовый лендинг для бренда спортивной одежды NEXFIT с упором на сложные скролл-анимации в духе Apple AirPods / Zentry / Lusion.

**Живая версия:** https://nexfit-nu.vercel.app

## Что показывает этот кейс

- **Пин-секция «Создан для движения»** — экран фиксируется при скролле, фотография раскрывается через `clip-path` (анимация на GPU, без дёрганий), текст проявляется по словам.
- **Гигантская типографика** — кинетический заголовок NEXFIT на весь экран в финале страницы.
- **Плавный скролл** — Lenis + GSAP ScrollTrigger, один scrub-таймлайн на весь сценарий пин-секции.
- **Адаптивные изображения** — `<picture>` с отдельными кадрами под портретную и альбомную ориентацию.
- Карточки товаров, отзывы со звёздным рейтингом, подвал с контактами.

## Технологии

- HTML / CSS / vanilla JS — без фреймворков и сборки
- [GSAP](https://gsap.com/) + ScrollTrigger — скролл-анимации
- [Lenis](https://lenis.darkroom.engineering/) — плавный скролл
- Хостинг — Vercel

## Как запустить

Сборка не нужна — это статический сайт:

```bash
# любой статический сервер, например:
python3 -m http.server 8080
# открыть http://localhost:8080
```

Или просто открыть `index.html` в браузере.

## Структура

```
index.html   — разметка
styles.css   — стили
app.js       — базовая логика (меню, слайдеры)
scroll.js    — скролл-анимации (Lenis + GSAP)
assets/      — изображения
```
