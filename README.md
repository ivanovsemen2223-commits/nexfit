# NEXFIT — лендинг спортивного костюма

🔗 Демо: https://nexfit-nu.vercel.app

Промо-страница одного товара — спортивного костюма. Сделана как презентация: цель — влюбить в продукт через движение, а не просто показать карточку. Целит в молодую активную аудиторию, которой важен вайб.

Главный момент — пин-секция «Создан для движения»: экран замирает, фотография раскрывается, текст выходит по словам, в финале имя NEXFIT растёт на весь экран. Тут человек залипает и запоминает бренд. Дальше идёт обычная карточка товара: галерея с переключением фото, выбор цвета и количества, табы «Описание / Характеристики / Уход», отзывы со звёздами. Фотки адаптивные — отдельные кадры под телефон и десктоп, чтобы костюм везде сидел красиво. Сценарий: проскроллил ради вау-эффекта, изучил товар, «добавил» в корзину.

**Тех:** HTML/CSS/JS, анимации на GSAP + ScrollTrigger и clip-path, плавный скролл Lenis.
